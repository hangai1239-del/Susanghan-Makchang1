A premium Apple-style Korean BBQ restaurant website.

STRUCTURE:
- index.html      : Main landing page
- menu.html       : Menu page  
- styles.css      : All styling
- script.js       : JavaScript functionality
- images/         : Folder for food images

HOW TO USE:

1. UNZIP the folder to your computer
2. OPEN index.html in any web browser
3. The website is fully functional!

HOW TO EDIT:

1. MENU ITEMS:
   - Open script.js
   - Find the 'menuItems' array
   - Add/edit/remove items as needed
   - Each item has: name, price, description, image path

2. IMAGES:
   - Add your food images to the 'images/' folder
   - Update the 'image' property in menuItems array
   - Recommended size: 800x600px, high quality

3. STYLES:
   - Open styles.css
   - Modify colors, fonts, spacing as needed
   - The design uses CSS variables for easy customization

4. CONTENT:
   - Edit text directly in index.html and menu.html
   - Update contact info, locations, etc.

FEATURES:

- Smooth animations
- Easy to customize
- No database needed

CONTACT INFO EDITING:
